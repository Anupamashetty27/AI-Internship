# Hiver AI Internship Assignment Submission

This repository contains the complete, production-ready solution for the Hiver AI Internship evaluation, covering Email Tagging (Part A), Sentiment Analysis Prompt Evaluation (Part B), and Mini-RAG System (Part C).

All parts of the solution were developed and evaluated using the **actual dataset** specified in the assignment (12 training emails, 60 evaluation emails, 4 real customers).
